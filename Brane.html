Wikipedia Scraper

This project was created as part of an internship for The Brane during the summer of 2018. Learn more about The Brane here!

The Brane is a start-up aiming to create a platform capable of displaying all the science and technology knowledge in the world. As a knowledge graph intern, I developed a Wikipedia scraper to produce a knowledge graph for Artificial Intelligence. My python script successfully extracts important information from thousands of Wikipedia articles and classifies according to The Brane's Knowledge Classification System. Given a starting category, the script can extract subcategories and URLs while still maintaining a close proximity to Artifical Intelligence related topics. The script outputs a CSV file containing nodes and links in order to create the graph structure. I expanded their database of topics by over 50%, adding thousands of nodes and developing algorithms to get thousands of links between topics.
