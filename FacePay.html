FacePay

This project was created for Hacktech 2018. See the devpost here!.

FacePay is a facial recognition payment system. Current payment systems are becoming obsolete. Nobody uses cash anymore and credit cards are easy to lose and a hassle to carry around. More and more people are switching to mobile payment platforms, yet there is still a big security risk here as mobile devices can easily be stolen or misplaced. FacePay allows you to pay for items using your face!

The system starts off by asking you to scan items and opens up the computer's camera. The user takes a picture of the item. We use the Amazon AWS Reckognition API to parse the text off the image. We filter out the text for words and pass it to the Walmart Search API. Using these keywords, we try to identify the product to determine how much it costs. The user must take a picture of every item. When they are ready to pay, they press a button and the system will ask the user to take a picture of their face. Using Microsoft's Cognitive Services API, we have a database of people which links to their face and phone number. We identify who is purchasing the items, ask for confirmation, and then send a receipt of the purchase using the Twilio API.

Our plans for the future are to add Amazon Alexa integration. We also plan to add biometric confirmation to ensure that the system is foolproof. For example, more testing will need to be done on people that look alike, such as twins.
