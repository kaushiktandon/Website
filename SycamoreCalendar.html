Sycamore Calendar

Sycamore Calendar is a web application that allows users to manage their Google Calendar. After users login to their Google accounts, they can insert and remove events to their calendar. They can search for their friends. If they are following their friend, they can insert events from that user's calendar. This web application provides an easy to use interface that makes it more convenient than directly using Google Calendar. I built this application using HTML, CSS, and JavaScript for the front-end, Java servlets in the backend, MySQL as the database, and Apache Tomcat as the web server.
